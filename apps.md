ğŸ“Š How It Works
The calculator uses a straightforward volumetric logic. Given a set of percentages $P_1, P_2, ... P_n$ and a target total volume $V_{total}$, it calculates the required volume for each solvent $V_s$ using the formula:

$$V_s = \frac{P_n}{100} \times V_{total}$$

 User Steps:
1.  Set Mode:Select "Percentage" mode.
2.  Input Ratios:Enter the percentage for each solvent (e.g., 12.5%, 43.5%, etc.).
3.  Define Total Volume:Enter the final amount of mixture you need (e.g., 750 mL).
4.  Calculate:Get an instant breakdown of every milliliter required.
